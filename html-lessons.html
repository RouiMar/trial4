üåê Introduction to HTML

The Foundation of Every Website

üéØ What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It's not a programming language - it's a markup language that defines the structure and content of a webpage.

üìÑ HTML Document Structure

The Blueprint of Every Web Page

üß± Building Blocks

Every HTML document follows the same basic structure. Think of it as a recipe that all browsers understand.

<!DOCTYPE html>

Defines the document type and HTML version

<html></html>

Root element that wraps all content

<head></head>

Contains meta information about the page

<body></body>

Contains all visible page content

üìù Complete HTML Template

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Welcome to my first webpage.</p>
</body>
</html>

üéØ Your First HTML File

Task: Create your first HTML file

  1. Open a text editor (Notepad, VS Code, etc.)
  2. Copy the template above
  3. Save as my-first-page.html
  4. Open it in a web browser

Congratulations! üéâ You just created a webpage!

üîß HTML Attributes

Adding Extra Information to Elements

üéØ What are Attributes?

Attributes provide additional information about HTML elements. They are always specified in the opening tag and come in name/value pairs like: name="value".

<a href="https://example.com" target="_blank">Visit Example</a>

In this example, href and target are attributes that modify how the link behaves.

üìã Common HTML Attributes

id

Unique identifier for an element

<div id="header">
class

CSS class for styling multiple elements

<p class="important">
src

Source file for images and scripts

<img src="photo.jpg">
alt

Alternative text for images

<img alt="A beautiful sunset">
href

Hyperlink reference for links

<a href="page.html">
style

Inline CSS styling

<p style="color: red;">

üîó Links & Images

Connecting and Visualizing Content

üåê Creating Links

The <a> (anchor) tag creates hyperlinks to other web pages or resources.

<!-- Basic link -->
<a href="https://example.com">Visit Example</a>

<!-- Link that opens in new tab -->
<a href="https://example.com" target="_blank">Open in New Tab</a>

<!-- Link to email -->
<a href="mailto:hello@example.com">Email Us</a>

<!-- Link to phone number -->
<a href="tel:+1234567890">Call Us</a>

üñºÔ∏è Adding Images

The <img> tag embeds images into your web page.

<!-- Basic image -->
<img src="photo.jpg" alt="A beautiful landscape">

<!-- Image with dimensions -->
<img src="photo.jpg" alt="A beautiful landscape" width="400" height="300">

<!-- Responsive image -->
<img src="photo.jpg" alt="A beautiful landscape" style="max-width: 100%;">
‚ö†Ô∏è

Important: Always Use Alt Text

The alt attribute is required for accessibility. It describes the image for screen readers and displays if the image fails to load.

üìä Tables

Organizing Data in Rows and Columns

üéØ Basic Table Structure

Tables are created with the <table> tag and contain rows (<tr>), header cells (<th>), and data cells (<td>).

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>30</td>
        <td>London</td>
    </tr>
</table>

üèóÔ∏è Advanced Table Features

For better structure, use <thead>, <tbody>, and <tfoot>.

<table>
    <caption>Employee Information</caption>
    <thead>
        <tr>
            <th>ID</th>
            <th>Name</th>
            <th>Department</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>001</td>
            <td>John Smith</td>
            <td>Marketing</td>
        </tr>
        <tr>
            <td>002</td>
            <td>Jane Doe</td>
            <td>Engineering</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Total Employees: 2</td>
        </tr>
    </tfoot>
</table>

üìù Forms

Collecting User Input

üéØ Basic Form Structure

Forms are created with the <form> tag and contain various input elements.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Submit">
</form>

üìã Common Form Elements

Text Input

<input type="text" name="username" placeholder="Enter username">

Password Input

<input type="password" name="password" placeholder="Enter password">

Textarea

<textarea name="message" rows="4" cols="50">Default text</textarea>

Select Dropdown

<select name="country">
    <option value="us">United States</option>
    <option value="ca">Canada</option>
    <option value="uk">United Kingdom</option>
</select>

Radio Buttons

<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>

<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

Checkboxes

<input type="checkbox" id="subscribe" name="subscribe" value="yes">
<label for="subscribe">Subscribe to newsletter</label>

üéØ Semantic HTML

Meaningful Structure for Better Web Pages

üß† What is Semantic HTML?

Semantic HTML uses elements that clearly describe their meaning to both browsers and developers. Instead of using generic <div> tags for everything, we use specific elements that describe their purpose.

üìö Book Analogy:

üìñ
Non-semantic

Just pages with text
No chapter titles or sections

üìö
Semantic

Clear structure with
chapters, headings, and sections

üèóÔ∏è Common Semantic Elements

<header>

Introductory content or navigation

<nav>

Navigation links

<main>

Main content of the document

<article>

Self-contained composition

<section>

Thematic grouping of content

<aside>

Content aside from main content

<footer>

Footer for document or section

üìù Semantic HTML Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Blog</title>
</head>
<body>
    <header>
        <h1>My Awesome Blog</h1>
        <nav>
            <a href="#home">Home</a>
            <a href="#about">About</a>
            <a href="#contact">Contact</a>
        </nav>
    </header>

    <main>
        <article>
            <h2>Understanding Semantic HTML</h2>
            <p>Semantic HTML makes your code more readable...</p>
        </article>

        <aside>
            <h3>Related Articles</h3>
            <ul>
                <li><a href="#">HTML Basics</a></li>
                <li><a href="#">CSS Fundamentals</a></li>
            </ul>
        </aside>
    </main>

    <footer>
        <p>© 2023 My Blog. All rights reserved.</p>
    </footer>
</body>
</html>
